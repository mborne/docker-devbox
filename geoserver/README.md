
# GeoServer (WMS, WFS, WMTS)

Docker container running [GeoServer](https://geoserver.org/)

## Usage with docker-compose

* Start GeoServer : `docker-compose up -d`

* Open http://geoserver.localhost/

* Login with admin/geoserver
