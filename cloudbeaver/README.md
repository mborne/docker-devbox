# cloudbeaver

Container running [dbeaver/cloudbeaver](https://hub.docker.com/r/dbeaver/cloudbeaver) official image.

## Usage with docker-compose

* Start cloudbeaver : `docker-compose up -d`
* Open https://cloudbeaver.dev.localhost/
* Follow instructions
