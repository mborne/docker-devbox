# CloudBeaver

Container running [dbeaver/cloudbeaver](https://hub.docker.com/r/dbeaver/cloudbeaver).

## Usage with docker

* Start cloudbeaver : `docker compose up -d`
* Open https://cloudbeaver.dev.localhost/
* Follow instructions
